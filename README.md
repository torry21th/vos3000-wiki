# VOS3000 Web 接口 Wiki

这是从 `VOS3000_webinterface.pdf` 生成的静态 Wiki 站点。

## 本地预览

```bash
python3 -m http.server 8765
```

然后打开：

```text
http://127.0.0.1:8765/
```

## GitHub Pages

仓库发布到 GitHub 后，在仓库的 `Settings -> Pages` 中选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

保存后访问 GitHub Pages 给出的地址即可。
